51  /**
52  * @brief Encapsulation of a parameter which can be used to configure Spectre
53  *
54  * Each Parameter instance represents a configuration option of Spectre. Depending on
55  * the Parameter its value can be specified using different data types which are all
56  * encapsulated in a ParameterVariant. Use the function Parameter::type() to get
57  * the expected type.
58  *
59  * Parameters can have a range of valid values. Use Parameter::hasBounds() to determine
60  * if a range is present. Values outside of the range cannot be set.
61  *
62  * If the function Parameter::isValid() returns false this instance is invalid. In most
63  * cases that means that the instance was retrieved through Configuration::getParameterByKey(),
64  * or Configuration::getParameterByName() but the requested Parameter is not supported by
65  * the actual Spectre instance.
66  */
