1 Overview {#mainpage}
2 ============
3 
4 Spectre is a software which allows to compute depth data from the raw-data
5 stream aquired from *PMD*'s time-of-flight (TOF) sensors. Main focus of the
6 project is to provide a flexible reference implementation of the depth data
7 processing pipeline optimized for different target platforms. The pipeline
8 offers a large set of parameters, and optional algorithms, so that it can be
9 parametrized to fit different use cases.
10 
11 Additionally, Spectre contains components for handling calibration data, and a
12 convenience C++ API. That is, Spectre is subdivided into the following three
13 main components:
14 
15 | Component | Purpose |
16 |----------------|-------------------------------------|
17 | libCalibration | Reading/Writing of calibration data |
18 | libProcessing | Depth data processing |
19 | libSpectre | Convenience API |
20 
21 This document describes the libSpectre API which wraps, and includes the other
22 two components.
