Overview

Spectre is a software which allows to compute depth data from the raw-data stream aquired from PMD's time-of-flight (TOF) sensors. Main focus of the project is to provide a flexible reference implementation of the depth data processing pipeline optimized for different target platforms. The pipeline offers a large set of parameters, and optional algorithms, so that it can be parametrized to fit different use cases.

Additionally, Spectre contains components for handling calibration data, and a convenience C++ API. That is, Spectre is subdivided into the following three main components:

Component Purpose
libCalibration Reading/Writing of calibration data
libProcessing Depth data processing
libSpectre Convenience API

This document describes the libSpectre API which wraps, and includes the other two components.

Supported Platforms

Currently, the following platforms are supported, and regularly tested:

  • Windows 7 (x86, AMD64)
  • Ubuntu Linux 14.04 (x86, AMD64)
  • Android 32 - Bit (ARMv7a with NEON VFPv4)
  • Android 64 - Bit (ARMv8a with NEON)
  • Linux ARM (Raspberry PI with NEON VFPv4)
  • Mac OSX (>= 10.10, AMD64)
