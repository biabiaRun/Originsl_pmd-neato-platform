89  /**
90  * @brief ISpectre is Spectre's main entry point, encapsulating calibration handling, and depth-data processing.
91  *
92  * Each instance of ISpectre is permanently coupled to a specific calibration, and intended for depth-processing
93  * of one raw data stream with specific settings. In contrast, all settings except the calibration can be changed
94  * during the lifetime of the object.
95  *
96  * The calibration which has been used to create a ISpectre instance determines the expected width/height of the
97  * input raw data. It will also return processed depth data with the same resolution. The used resolution can be
98  * requested through the ISpectre::getWidth() and ISpectre::getHeight() methods.
99  */
